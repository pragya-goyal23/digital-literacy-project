Digital Literacy Portfolio project covering infographic, profiles, platforms, email etiquette and cyber awareness tasks.

--> Overview

This project was created as part of the Digital Literacy course. It includes tasks related to digital awareness, online platforms, communication, and cyber safety. The aim of this project is to understand how to use digital tools responsibly and maintain a safe and professional online presence.

---

--> Tasks Completed

--> Task 1 – Infographic
Created a digital literacy infographic using Canva covering safe internet practices, useful tools, and responsible online behavior.

--> Task 2 – Digital Portfolio
Set up and updated profiles on platforms like GitHub, LinkedIn, and Kaggle to build a basic professional online presence.

--> Task 3 – Platforms
Explored HackerRank and created a Google Form quiz related to digital literacy.

--> Task 4 – Email Etiquette
Drafted professional emails and created a social media do’s and don’ts checklist.

--> Task 5 – Cybercrime
Prepared a phishing case study and a prevention checklist for online safety.
